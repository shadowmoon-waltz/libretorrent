# 機能

* DHT、PeX、暗号化、LSD、UPnP、NAT*PMP、μTP
* IP フィルタリング (eMule dat と PeerGuardian)
* チューニング機能 (ネットワーク設定、電源管理、バッテリ制御、UI 設定など)
* 大量のファイルと大きなファイルを含むトレントをサポート
* HTTP\S とマグネットリンクのサポート
* トラッカーとピアのプロキシをサポート
* ダウンロード中にファイルを移動する機能
* ダウンロード終了時に別のディレクトリや外付けドライブへのファイルの自動移動が可能
* ファイルとフォルダの優先順位を指定する機能
* ダウンロードするファイルを選択する機能
* 連続的にダウンロード可能
* マテリアルデザイン
* ダーク/ブラックテーマ
* タブレットに最適化された UI
* スケジューリング
* RSS マネージャー
* Android TV 対応
* トレント作成機能
* 選択したファイルのストリーミング
* その他多数
